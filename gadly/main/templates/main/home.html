GADLY

A Gender Based Sensitive Paraphrasing Tool

The purpose of this website is to present a detailed description of the web-based application titled GADly: A Gender-Based Paraphrasing Tool. It will explain the purpose and features of the system, the interfaces of the system, what the system will do, the constraints under which it must operate, and how the system will react to data that is given to it. This document is intended for both the stakeholders and the developers of the system and will be proposed to the Commission on Higher Education Regional Office Region IV-B for its approval.The following are the coders:
